Welcome to website portfolio,this website build by me alone with the help of the open source AI copilot i worked heavly on the back-end while the AI focesed on fixing errors and handuling the front-end

we have engineered a robust Django-based application that goes far beyond basic functionality. From the ground up

Technologies & Tools

  • Django Framework: The backbone of the application, powering dynamic views, models, and secure user interactions.
  • Bootstrap 5: For responsive, mobile-friendly design with clean UI components and modern styling.
  • Bootstrap Icons: Used extensively for intuitive visual feedback (e.g., star ratings, buttons).
  • Custom JavaScript: Enhancing interactivity — including dynamic image previews, star-based review input, and collapsible forms.
  • Windows Batch Automation: I helped design universal batch scripts that:
    • Detect Python installations
    • Create virtual environments
    • Install dependencies
    • Launch Django servers

Features I Build

  • Interactive Product Gallery: Clickable thumbnails with dynamic image switching.
  • Star-Based Review System: Users rate products using intuitive star icons
  • Review Form Toggle: Collapsible review input that adapts based on whether the user has already submitted feedback.
  • Average Rating Display: Real-time visual representation of product ratings using filled and empty stars.
  • Portable Setup Scripts: Batch files that make onboarding seamless for any developer — no manual setup required.
